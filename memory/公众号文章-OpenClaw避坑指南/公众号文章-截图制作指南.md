# 公众号文章截图制作指南

## 需要截图的 2 处代码块

### 截图 1：安装 Memory 插件

**截图内容：**
```bash
# 1. 安装 memory-core 插件
openclaw plugin install memory-core

# 2. 启用记忆搜索
# 在 openclaw.json 中设置：
"memorySearch": { "enabled": true }
```

**截图要求：**
- 背景：深色终端（推荐）
- 字体：清晰可读
- 宽度：800px 左右
- 格式：PNG 或 JPG

---

### 截图 2：飞书安全配置

**截图内容：**
```json
"groupPolicy": "allowlist",
"groupAllowlist": ["oc_xxx", "oc_yyy"],
"groupAllowFrom": ["ou_你的用户ID"]
```

**截图要求：**
- 背景：VSCode 或代码编辑器
- 高亮：JSON 语法高亮
- 宽度：800px 左右
- 格式：PNG 或 JPG

---

## 截图工具推荐

| 工具 | 快捷键 | 特点 |
|------|--------|------|
| Mac 自带截图 | Cmd+Shift+4 | 选区截图，自动保存桌面 |
| iShot | 自定义 | 带标注、长截图 |
| Xnip | 自定义 | 精美阴影效果 |

---

## 截图后处理

1. **裁剪**：去掉多余空白
2. **压缩**：单张 < 2MB
3. **命名**：
   - `screenshot-01-memory.png`
   - `screenshot-02-security.png`

---

## 发表步骤

1. 复制 `公众号文章-OpenClaw避坑指南-纯文本版.txt` 内容
2. 粘贴到公众号编辑器
3. 在标记位置插入截图
4. 调整文字格式（加粗、标题）
5. 设置封面图
6. 手机预览
7. 发表
