## 录制建议

### 文案1（AI团队）
- **风格**：兴奋、展示成果
- **背景**：电脑屏幕展示多Agent协作界面
- **服装**：休闲商务
- **最佳发布时间**：晚上8-10点

### 文案2（国内外差距）
- **风格**：严肃、思考者
- **背景**：书架或简洁墙面
- **服装**：衬衫或简约T恤
- **最佳发布时间**：中午12-1点

### 文案3（别写周报）
- **风格**：犀利、直接
- **背景**：办公桌面
- **服装**：日常休闲
- **最佳发布时间**：早上7-9点

### 通用技巧
1. **前3秒**必须抓人，用震惊表情或反常识观点
2. **每15秒**一个转折点，保持节奏
3. **数据**要具体（4小时、50个视频、179万播放）
4. **结尾**必须有钩子，引导互动
5. **字幕**要大，关键数据标红
6. **BGM**用轻快的电子音乐
