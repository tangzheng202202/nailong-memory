# 广州天河区星辉中心烟商贷陌拜作战手册

## 一、目标客户分析

### 1.1 星辉中心周边烟酒商行分布
**区域特点：**
- 天河区核心商圈，写字楼密集
- 周边有多个高档小区
- 餐饮娱乐场所集中
- 人流量大，消费能力强

**目标客户画像：**
| 维度 | 特征 |
|------|------|
| **店铺类型** | 烟酒店、酒类专卖店、便利店附带烟酒区 |
| **经营年限** | 1年以上（有稳定流水） |
| **月流水** | 10万以上 |
| **痛点** | 资金周转、旺季备货、扩大经营 |
| **决策人** | 店主/老板娘（通常直接决策） |

### 1.2 高潜力客户特征
✅ **优先拜访：**
- 店面装修较新（有投资意愿）
- 陈列高档烟酒（客单价高）
- 有"茅台""五粮液"等名酒展示（资金需求大）
- 门口有"回收烟酒"招牌（资金流动性需求）
- 正在装修或扩大店面

❌ **低优先级：**
- 店面老旧、陈列杂乱
- 只卖普通香烟（利润薄）
- 店主年龄过大（接受度低）

## 三、营销话术

### 3.1 开场白（5秒内抓住注意力）

**版本A（直接型）：**
> "老板您好，我是广发银行的，专门做烟酒商户的无抵押贷款，最高50万，3分钟了解一下？"

**版本B（痛点型）：**
> "老板，最近茅台进货需要资金周转吗？我们有个专门针对烟酒行的贷款产品，不用抵押，随借随还。"

**版本C（数据型）：**
> "老板您好，我们银行刚推出烟商贷，天河区已经有XX家烟酒行批了，最高50万，月息只要X厘，您有兴趣了解一下吗？"

### 3.2 产品介绍（30秒讲清楚）

**核心卖点：**
```
广发烟商贷 = 专为烟酒行设计
├─ 额度：最高50万
├─ 利率：月息X厘（年化X%）
├─ 期限：最长3年
├─ 还款：随借随还，按日计息
├─ 担保：纯信用，无抵押
└─ 放款：最快当天到账
```

**对比优势：**
| 对比项 | 烟商贷 | 传统贷款 | 民间借贷 |
|--------|--------|----------|----------|
| 抵押 | 不需要 | 需要房产 | 不需要 |
| 利率 | 低 | 低 | 高（2-3分） |
| 速度 | 快（当天） | 慢（1-2周） | 快 |
| 额度 | 适中 | 高 | 低 |

### 3.3 应对常见拒绝

**拒绝1："不需要，我资金充足"**
> "理解理解，老板生意好。不过我们这个是备用金性质，不用不产生利息，万一哪天旺季备货或者遇到好货源，随时能用上。而且额度批下来不用也不影响，就当多个选择。"

**拒绝2："利率太高"**
> "老板，我们这个月息X厘，年化X%，您对比一下民间借贷（2-3分），其实便宜很多。而且我们是银行产品，正规安全，没有任何隐形费用。"

**拒绝3："手续太麻烦"**
> "老板，我们烟商贷就是简化流程，只需要身份证、营业执照、银行卡流水，最快当天就能放款。您要是现在方便，我帮您看看资料，5分钟就能知道能批多少额度。"

**拒绝4："我再考虑考虑"**
> "没问题老板，您考虑是应该的。这样，我加您个微信，把产品资料发给您，您有空看看。我们这个月有活动，批额度的客户送XX礼品，您考虑好了随时联系我。"

**拒绝5："已经有其他银行贷款了"**
> "那说明您信用好，银行愿意贷。我们广发烟商贷可以作为补充，额度不冲突。很多老板都是几家银行都有额度，哪家利率低用哪家，灵活调配。"

### 3.4 促成话术

**紧迫感：**
> "老板，这个月我们针对天河区烟酒行有专项额度，利率比平时低X厘，过了这个月就恢复原价了。"

**从众心理：**
> "老板，星辉中心这边已经有3家烟酒行批了我们的贷款，您隔壁的XX店上周刚放了20万。"

**低门槛试探：**
> "老板，要不这样，我先帮您查一下额度，不查征信，就是预估一下，看看能批多少，您再决定要不要，怎么样？"

## 四、陌拜技巧

### 4.1 最佳拜访时间
| 时间段 | 适合度 | 原因 |
|--------|--------|------|
| 9:30-10:30 | ⭐⭐⭐⭐ | 刚开门，老板在整理，有时间 |
| 10:30-11:30 | ⭐⭐⭐ | 开始有客人，但不太忙 |
| 14:00-15:00 | ⭐⭐⭐⭐⭐ | 午休后，老板精神好 |
| 15:00-16:30 | ⭐⭐⭐ | 下午营业高峰前 |
| 17:00后 | ⭐⭐ | 准备下班，心思不在 |

**避开时间：**
- 中午12:00-14:00（吃饭休息）
- 晚上18:00后（下班）

### 4.2 形象与物料

**着装：**
- 商务休闲（衬衫+西裤，不打领带）
- 佩戴广发银行工牌
- 背商务包（装资料）

**必带物料：**
- [ ] 广发银行名片（20张）
- [ ] 烟商贷产品折页
- [ ] 申请表（空白）
- [ ] 计算器（算利息）
- [ ] 小礼品（打火机、纸巾盒等带广发logo）
- [ ] 笔记本（记录客户信息）

### 4.3 陌拜流程

**Step 1：观察（30秒）**
- 店面大小、装修档次
- 陈列的烟酒品牌
- 店内客流情况
- 判断决策人（谁在收银台）

**Step 2：开场（10秒）**
- 微笑进门
- 直接表明身份和来意
- 观察老板反应

**Step 3：需求挖掘（2-3分钟）**
- "老板这店开了多久了？"
- "生意怎么样，主要是零售还是批发？"
- "最近有没有扩大经营的打算？"
- "资金周转方面有没有压力？"

**Step 4：产品介绍（2-3分钟）**
- 根据需求点介绍产品
- 重点讲能解决什么痛点
- 用数据说话（利率、额度、速度）

**Step 5：促成或留资（1-2分钟）**
- 尝试现场申请或预约
- 加微信发资料
- 约定下次拜访时间

**Step 6：记录（离开后）**
- 店铺名称、地址
- 老板姓名、电话
- 需求程度评级（A/B/C）
- 下次跟进时间
