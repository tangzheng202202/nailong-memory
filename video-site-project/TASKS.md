# 视频网站项目任务分配

## 项目概述
创建一个简单的视频网站，支持上传、播放、列表展示。

## 任务分配

### 任务 1: 前端开发 (Claude Agent)
**工作目录**: ../workspace-claude
**任务**: 
- 创建 index.html - 首页（视频列表 + 上传按钮）
- 创建 player.html - 播放页面
- 使用现代 CSS (Flexbox/Grid)
- 响应式设计

### 任务 2: 后端 API (Codex Agent)
**工作目录**: ../workspace-codex
**任务**:
- 创建 server.js - Express 服务器
- API: GET /api/videos - 获取视频列表
- API: POST /api/upload - 上传视频
- API: GET /api/video/:id - 获取视频信息
- 使用内存存储（简化版）

### 任务 3: 测试与文档 (Creative Agent)
**工作目录**: ../workspace-gemini
**任务**:
- 创建 README.md - 项目说明
- 创建 API.md - API 接口文档
- 创建 test.md - 测试用例说明
